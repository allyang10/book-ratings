My Top Fantasy Pick:

Genre: Fantasy

Author: Patrick Rothfuss

Page Count: 665

# of Books: 3 (third one is probably never coming out so don't read if you don't like an unfinished series)

Read this series if you like:

- A genius protagonist

- Beautiful writing

- A very interesting magic system and characters

- Multiple smaller stories instead of one larger plot
